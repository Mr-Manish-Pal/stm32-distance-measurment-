# ⭐ About This Project

This project demonstrates the complete workflow of programming the **STM32F103C8T6 Blue Pill** using **ST-Link V2** and **STM32CubeIDE**.

Unlike simple "LED Blink" examples, this repository focuses on the complete firmware development workflow used by embedded engineers.

The repository has been structured to help

- Engineering Students
- Beginners in Embedded Systems
- Firmware Developers
- Electronics Enthusiasts
- Placement Preparation
- Technical Interviews

# 🚀 Quick Start

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/STM32-Programming-Blue-Pill.git
```

Open the project in **STM32CubeIDE**

Connect **ST-Link V2**

Build the project

Flash the firmware

Verify the output
